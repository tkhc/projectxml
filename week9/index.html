Summary

I found that writing out the queries was not the most time consuming part but evaluating the queries was. Since I am using visual studio code extensions for all my previous tutorials, I thought I could find something that work with this tutorial too. I downloaded two extensions (XML Tools and MarkLogic Developer Tools), neither of them work. Then I realized I was missing the XQuery execution engine. I didn't want to pay for Altova (the website is also confusing to look at), I ended up downloading BaseX instead. The engine still didn't work with the extension but luckily BaseX came with a GUI. All I have to do is open the files and hit execute, it would show me the status of the query. If there is an error, it would display the error. BaseX really helped me to see where the typos or syntax errors were and the rest of process was very smooth.
